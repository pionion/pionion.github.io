🌍 ION is a distributed real-time communication system, the goal is to communicate anydevice, anytime, anywhere! 🌍

☕️ION Modules

  • Backend built with Pion and Golang

    • ion-sfuis a SFU which support GRPC and JsonRPC

    • ion-avp is a realtime Audio/Video processing service

    • ion contains biz and islb which support signal and biz logic

  • Frontend and app build with H5/JS/Flutter and Flutter-WebRTC

    • ion-sdk-jsis a ts sdk which support your web app communicate to ion-sfu

    • ion-sdk-flutteris a flutter sdk which support your mobile/pc app communicate to ion-sfu
