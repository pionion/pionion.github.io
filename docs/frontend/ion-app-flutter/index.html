3. ion-app-flutter

1. Project​

https://github.com/pion/ion-app-flutter

2. Getting Started​

Make sure your flutter is 2.0.

  • ./scripts/project_tools.sh create

3. For Android/iOS​

First, attach your device to pc, find your device:

  • flutter devices

Then:

  • flutter run -d {yourdevice}

4. For macOS​

  • flutter run -d macos

5. For Web​

  • flutter run -d chrome
